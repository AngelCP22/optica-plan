# Preview de diseno - Oftal Medic

Esta carpeta contiene una maqueta estatica para presentar el diseno en GitHub Pages.

## Como verlo localmente

Abrir `index.html` en el navegador o levantar un servidor local apuntando a esta carpeta.

## Como publicarlo en GitHub Pages

1. Subir el repositorio a GitHub.
2. Entrar a `Settings > Pages`.
3. En `Build and deployment`, elegir `Deploy from a branch`.
4. Seleccionar la rama principal y la carpeta `/docs`.
5. Abrir la URL que entrega GitHub Pages y entrar a `/design-preview/`.

## Alcance

- Es solo presentacion visual.
- No usa base de datos.
- No expone pacientes reales.
- No modifica la logica de Laravel.
